## Live Operational Files

The following files support daily operational workflows and remain at the repository root:
- `tasks.md` - Current day's task list (updated daily by Claude)
- `task-register.md` - Live progress tracking (updated by Codex)  
- `session-handoff.md` - Latest session context
- `archive/` - Historical task and session records

These operational files are part of the [Development Process v1.1](development/processes/) framework.
